# AGENTS

Ce dépôt contient une application web composée d'un serveur Node.js qui expose des flux audio et d'une interface statique.

- Installez les dépendances avec `npm install`, puis lancez le serveur avec `npm start`.
- Il n'existe pas de suite de tests automatisés.
- Rédigez les messages de commit en français.
- Résumez brièvement les modifications dans la description de la Pull Request.
